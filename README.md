# 8-Bit Mario Bros Game

A classic 8-bit style Mario Bros game built with HTML5, CSS3, and vanilla JavaScript.

![Game Screenshot](https://github.com/user-attachments/assets/19f8f2c6-9e7c-4594-8cb2-09a55c2c1451)

## Features

- **8-bit Style Graphics**: Authentic retro pixel art design
- **Classic Mario Character**: Pixel-perfect Mario with red cap, blue overalls, and brown shoes
- **Platform Gameplay**: Jump across multiple platforms with realistic physics
- **Enemies**: Goombas that patrol platforms and can be defeated by jumping on them
- **Collectibles**: Golden coins to increase your score
- **Lives System**: 3 lives to complete the level
- **Score Tracking**: Earn points by defeating enemies (100 pts) and collecting coins (50 pts)
- **Responsive Controls**: Smooth keyboard controls for movement and jumping

## How to Play

1. Open `index.html` in a web browser
2. Use **Arrow Keys** (‚Üê ‚Üí) to move Mario left and right
3. Press **SPACE** to jump
4. Jump on enemies to defeat them and earn points
5. Collect coins for extra points
6. Avoid getting hit by enemies (costs 1 life)
7. Press **R** to restart the game

## Game Elements

- **Mario**: The hero character you control
- **Platforms**: Brown brick platforms to jump on
- **Goombas**: Brown enemy creatures that patrol platforms
- **Coins**: Golden collectibles worth 50 points each
- **Clouds & Bushes**: Decorative background elements

## Scoring

- Defeat a Goomba: 100 points
- Collect a coin: 50 points

## Files

- `index.html` - Main HTML structure
- `style.css` - Game styling and layout
- `game.js` - Game logic and mechanics

## Technologies Used

- HTML5 Canvas
- CSS3
- Vanilla JavaScript (ES6+)
